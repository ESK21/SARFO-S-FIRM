# USB-C 5V Power Supply

# Overview

> Designed using EasyEDA | Focus: Compact, reliable 5V USB-C power delivery

## Overview
A compact USB-C based 5V power supply designed for stable low-voltage output applications.

## Key Features
- USB-C CC1/CC2 configuration using 5.1kΩ resistors
- Supports standard USB-C current negotiation
- 5V output
- LED power indication
- Decoupling capacitors for noise filtering

## Design Decisions
- Used 5.1kΩ resistors to signal default current mode
- Added capacitors for voltage stability
- Designed compact PCB layout for practical use
