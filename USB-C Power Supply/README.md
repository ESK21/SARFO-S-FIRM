# USB-C 5V Power Supply

## Overview
This project is a USB-C-based 5V power supply designed using EasyEDA.

## Key Features
- USB-C CC1/CC2 configuration using 5.1k resistors
- Stable 5V output
- LED power indication
- Decoupling capacitors for noise filtering

## Design Considerations
- Proper USB-C current negotiation
- Power stability and filtering
- Simple and cost-effective design
